Configurer la clé API Gemini

L'outil utilise Gemini Flash de Google — 1 500 requêtes gratuites par jour, sans carte bancaire. Voici comment obtenir votre clé en 2 minutes :

  1. Rendez-vous sur aistudio.google.com/app/apikey
  2. Connectez-vous avec votre compte Google
  3. Cliquez sur Create API Key → Copiez la clé
  4. Collez-la ci-dessous et cliquez sur Démarrer
🔒 La clé est stockée uniquement dans votre navigateur (localStorage). Elle n'est jamais envoyée ailleurs qu'à l'API Google.
